# 07_Prompts

## Purpose
A curated prompt engineering library enabling consistent, rule-compliant AI-assisted BA/PO/QA output.

## Scope
Covers discovery, BRD, user story, gap analysis, and process flow prompt patterns.

## Intended Usage
Copy and adapt the relevant prompt, always pairing it with source material and the applicable standard from [02_Standards](../02_Standards/README.md).
