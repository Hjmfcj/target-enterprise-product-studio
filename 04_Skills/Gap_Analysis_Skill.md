# Gap Analysis Skill

## Purpose
Executable skill for identifying missing requirements, rules, stakeholders, validations, approvals, integrations, or reports before a deliverable is finalized.

## Scope
Applies whenever the matching task type is requested in any project.

## Intended Usage
Runs the checklist defined in [00_System/05_Quality_Gates.md](../00_System/05_Quality_Gates.md) against any draft deliverable.

Will be extended with input/output examples as more projects are onboarded.
