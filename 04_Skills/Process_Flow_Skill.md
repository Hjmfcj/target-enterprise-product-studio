# Process Flow Generation Skill

## Purpose
Executable skill for producing Mermaid/BPMN process flow diagrams from documented workflows.

## Scope
Applies whenever the matching task type is requested in any project.

## Intended Usage
Generates AS-IS and TO-BE flowcharts strictly from source-confirmed process steps and decision points.

## Future Expansion
Will be extended with input/output examples as more projects are onboarded.
