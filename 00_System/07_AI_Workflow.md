# AI Workflow

## Purpose
Describes how an AI assistant should navigate and apply this repository when supporting discovery, documentation, or delivery work.

## Scope
Applies to any AI-assisted session (e.g. Claude, Copilot) operating against this repository.

## Intended Usage
1. Load [01_System_Instructions.md](01_System_Instructions.md) and [04_Project_Rules.md](04_Project_Rules.md) first.
2. Identify the relevant role in [01_Roles](../01_Roles/README.md).
3. Select the applicable standard from [02_Standards](../02_Standards/README.md).
4. Apply the relevant framework from [03_Frameworks](../03_Frameworks/README.md) and skill from [04_Skills](../04_Skills/README.md).
5. Use a template from [05_Templates](../05_Templates/README.md) as the output shell.
6. Validate against [08_Checklists](../08_Checklists/README.md) before delivery.
7. Store the finished artifact under the relevant project in [10_Projects](../10_Projects/README.md).

## Future Expansion
Will be converted into an executable orchestration script once the repository is wired into an agentic pipeline.
