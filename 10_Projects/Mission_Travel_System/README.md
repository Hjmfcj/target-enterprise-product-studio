# MTN Sudan Travel Mission Management System

## Purpose
This is the project workspace for the MTN Sudan Travel Mission Management System engagement, delivered by Target Integrated Systems on the Microsoft 365 / SharePoint platform. All project knowledge is maintained here as a single source of truth.

## Scope
Covers the complete project lifecycle from Discovery through Handover for the MTN Sudan Travel Mission Management System — a platform managing the full employee travel mission lifecycle: creation, approval, GL review, treasury payment, extension, closure, and settlement.

## Intended Usage
Begin from [Project_Metadata.md](Project_Metadata.md) for all identifying context, then navigate to the relevant phase folder. AI assistants should read Project_Metadata first.
