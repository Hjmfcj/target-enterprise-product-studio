# 06_Playbooks

## Purpose
Provides operational, step-by-step playbooks for recurring BA/PO/QA activities.

## Scope
Covers discovery workshops, stakeholder interviews, backlog grooming, and UAT preparation.

## Intended Usage
Follow the relevant playbook end-to-end during the matching activity; pair with the corresponding template from [05_Templates](../05_Templates/README.md).
