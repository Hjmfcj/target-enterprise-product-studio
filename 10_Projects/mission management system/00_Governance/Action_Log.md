# Action Log

## Purpose
A running, append-only log of action items generated during this project, so that ownership and due dates are trackable independently of any single meeting's minutes.

## Scope
Applies to this specific project workspace only. Covers action items from any phase of the engagement.

## Intended Usage
Add a new row per action item as it is raised (typically from a MOM entry). Do not remove rows once closed — update Status instead. Mark unavailable content as `MISSING INFORMATION`.

**Fields / Sections:** Action ID | Description | Owner | Raised Date | Due Date | Source (Related MOM/Decision Entry) | Status (Open/In Progress/Closed/Overdue) | Notes

Will be refined based on recurring client feedback on deliverable structure.
