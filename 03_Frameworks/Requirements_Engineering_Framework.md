# Requirements Engineering Framework

## Purpose
Provides a structured lifecycle from raw discovery input to validated, traceable requirements.

## Scope
Applies to all projects listed in [10_Projects](../10_Projects/README.md).

## Intended Usage
Use as the step-by-step method during the relevant phase of an engagement.

**Key elements:** Elicitation → Analysis → Specification → Validation → Traceability, per [00_System/04_Project_Rules.md](../00_System/04_Project_Rules.md).

## Future Expansion
Will be refined with lessons learned from completed engagements.
