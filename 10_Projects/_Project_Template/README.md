# _Project_Template

## Purpose
A blank project workspace shell to be copied for every new engagement.

## Scope
Provides the standard subfolder layout expected inside every project workspace.

## Intended Usage
Copy this entire folder, rename it to the project name, complete [Project_Metadata.md](Project_Metadata.md) first, then begin populating each subfolder as discovery progresses. [00_Governance](00_Governance/README.md) is updated continuously across all phases rather than being tied to a single one.
