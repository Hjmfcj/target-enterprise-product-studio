# 99_Archive

## Purpose
Stores deprecated or superseded content that should not be deleted for historical/audit purposes.

## Scope
Applies to any file from any other folder that has been formally superseded.

## Intended Usage
Move superseded files here with their original relative path preserved where practical, and record the reason in [Archive_Policy.md](Archive_Policy.md)'s log.

## Contents
- [Archive Policy](Archive_Policy.md)

## Future Expansion
Will grow indefinitely; periodic review may relocate very old material to cold storage outside the repository.
