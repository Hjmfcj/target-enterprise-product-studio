# 03_Frameworks

## Purpose
Contains repeatable methodological frameworks that structure how discovery, requirements, delivery, architecture, and risk work is performed.

## Scope
Applies across all projects; frameworks are methodology, not project-specific content.

## Intended Usage
Apply the relevant framework as the methodological backbone before using templates or prompts to produce actual deliverables.
