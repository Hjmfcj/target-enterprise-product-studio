# Archive Policy

## Purpose
Defines when and how content is moved into the Archive rather than deleted.

## Scope
Applies to all folders in this repository.

## Intended Usage
- Content is archived, never deleted, to preserve audit history.
- When archiving, add an entry below: file, original location, date archived, reason.
- Archived content is excluded from active navigation links in folder README.md files.
