# Clarification Register Template

## Purpose
A ready-to-fill register for tracking open clarification questions raised during discovery or backlog refinement.

## Scope
Applies to any project requiring this deliverable type.

## Intended Usage
Copy this file into the target project folder and complete each field. Do not remove sections — mark unavailable content as `MISSING INFORMATION`.

**Fields / Sections:** Question ID | Question | Raised By | Date Raised | Area/Module | Status | Answer | Answered By | Date Answered

## Future Expansion
Will be refined based on recurring client feedback on deliverable structure.
