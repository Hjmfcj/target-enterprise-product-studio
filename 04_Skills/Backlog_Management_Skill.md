# Backlog Management Skill

## Purpose
Executable skill for organizing requirements into the Epic → Feature → User Story → Task hierarchy.

## Scope
Applies whenever the matching task type is requested in any project.

## Intended Usage
Follows the [Azure DevOps Standard](../02_Standards/Azure_DevOps.md); classifies by priority, business value, complexity, and dependencies.
