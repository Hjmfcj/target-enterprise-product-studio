# 02_Standards

## Purpose
Houses the enterprise documentation and engineering standards that every deliverable produced from this repository must follow.

## Scope
Covers BRD, FRD, User Stories, Acceptance Criteria, Azure DevOps, Naming, API, UI/UX, and Security standards.

## Intended Usage
Reference the relevant standard before authoring any deliverable; link back to the standard from the deliverable itself for traceability.
