# UX Architect

## Purpose
Defines the end-to-end user experience architecture, information architecture, and interaction models.

## Scope
Applies within the UX Architect function across all projects tracked in [10_Projects](../10_Projects/README.md).

## Intended Usage
Used when designing screen flows, navigation structures, and interaction patterns.

## Core Responsibilities
- Produces screen inventories, user flows, wireframe-level structure, and interaction specifications.
