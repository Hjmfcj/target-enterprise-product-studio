# User Story Prompts

## Purpose
A library of prompts for generating TFS/Azure DevOps-ready User Stories.

## Scope
Applies whenever the matching AI-assisted task is performed.

## Intended Usage
Adapt the example prompt below to the specific source material at hand.

**Example Prompt:**
> "Convert the following functional requirements into User Stories with Given/When/Then Acceptance Criteria, Priority, Dependencies, and Business Rules. Do not invent acceptance criteria not derivable from the source."

## Future Expansion
Will be expanded with additional prompt variants as usage patterns emerge.
