# Naming Convention Standard

## Purpose
Defines file, folder, and work-item naming conventions across the repository and backlog tools.

## Scope
Applies to all Markdown files, Azure DevOps items, and Figma files referenced by this repository.

## Intended Usage
Files: Title_Case_With_Underscores.md. Folders: NN_Category_Name. Work items: [Module] - Short descriptive title.

## Future Expansion
Will be versioned as client-specific variations emerge; deviations must be documented, not silently applied.
