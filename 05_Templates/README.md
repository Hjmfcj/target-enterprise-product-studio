# 05_Templates

## Purpose
Provides ready-to-use, standard-compliant document shells so every new deliverable starts from a consistent structure.

## Scope
Covers BRD, FRD, User Story, MOM, RTM, UAT, PID, and Clarification Register templates.

## Intended Usage
Copy the relevant template into the appropriate project folder under [10_Projects](../10_Projects/README.md) and fill it in using only source-confirmed information.
