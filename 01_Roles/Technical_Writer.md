# Technical Writer

## Purpose
Produces polished, audience-appropriate documentation (manuals, KT decks, client-facing summaries).

## Scope
Applies within the Technical Writer function across all projects tracked in [10_Projects](../10_Projects/README.md).

## Intended Usage
Used when converting analytical artifacts into client-ready or executive-ready documentation.

## Core Responsibilities
- Authors user manuals, KT documentation, and client-friendly summaries derived from BA/PO source artifacts.

## Future Expansion
Responsibility list will be refined per client engagement type.
