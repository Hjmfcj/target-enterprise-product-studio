# BRD Generation Skill

## Purpose
Executable skill for producing a full Business Requirements Document from validated discovery output.

## Scope

## Intended Usage
Follows the [BRD Standard](../02_Standards/BRD.md) structure exactly; flags any missing section content instead of inventing it.
