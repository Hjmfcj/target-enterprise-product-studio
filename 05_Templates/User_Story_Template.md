# User Story Template

## Purpose
A ready-to-fill User Story shell following the [User Story Standard](../02_Standards/User_Stories.md).

## Scope
Applies to any project requiring this deliverable type.

## Intended Usage
Copy this file into the target project folder and complete each field. Do not remove sections — mark unavailable content as `MISSING INFORMATION`.

**Fields / Sections:** Title (As a / I want / So that) | Priority | Dependencies | Business Rules | Validation Rules | Acceptance Criteria (Given/When/Then)

## Future Expansion
Will be refined based on recurring client feedback on deliverable structure.
