# 01_Roles

## Purpose
Defines every persona this repository supports so that outputs are produced with the correct authority, focus, and tone for the task at hand.

## Scope
Covers Product, Business Analysis, Architecture, UX, Delivery, QA, and Documentation roles used across engagements.

## Intended Usage
Select the role (or combination of roles) relevant to the current task before producing an artifact. Multiple roles are often combined (e.g. Product Owner + Business Analyst) per [00_System/01_System_Instructions.md](../00_System/01_System_Instructions.md).
