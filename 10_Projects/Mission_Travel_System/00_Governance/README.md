# Governance

## Purpose
Holds the persistent, cross-phase governance record for this project — meeting minutes, decisions, risks/assumptions/issues/dependencies, and action items — accumulated continuously from kickoff through closure.

## Scope
Applies to this specific project workspace only. Unlike `01_Discovery` through `06_Handover`, this folder is not tied to a single phase; it is updated throughout the entire engagement.

## Intended Usage
Update the relevant log after every meeting, decision point, or risk/action identification, regardless of which phase the project is currently in. Reference entries from these logs in downstream deliverables (BRD Risks/Assumptions/Dependencies sections, UAT sign-off, Handover) rather than duplicating them.

## Future Expansion
Will accumulate entries for the life of the engagement; retired/closed items are struck through or status-marked rather than deleted, consistent with this repository's no-delete archive philosophy.
