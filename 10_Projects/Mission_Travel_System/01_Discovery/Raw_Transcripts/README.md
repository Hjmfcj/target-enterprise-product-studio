# Raw Transcripts

## Purpose
Holds unedited source transcripts from all client and internal sessions, retained verbatim for traceability. These files are the evidentiary foundation from which meeting minutes and requirements are derived.

## Scope
Applies to this specific project workspace only.

## Intended Usage
Use only as a traceability source when verifying a specific claim in a distilled artifact (MOM, BRD, Clarification Register). Do not treat raw transcripts as confirmed requirements — only items synthesised into a distilled MOM or the approved source-of-truth document carry requirement authority. See [../Discovery_Notes.md](../Discovery_Notes.md) for the mapping of each transcript to its corresponding meeting.
