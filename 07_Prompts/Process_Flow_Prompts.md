# Process Flow Prompts

## Purpose
A library of prompts for generating Mermaid process flow diagrams from documented workflows.

## Scope
Applies whenever the matching AI-assisted task is performed.

## Intended Usage
Adapt the example prompt below to the specific source material at hand.

**Example Prompt:**
> "Generate a Mermaid flowchart (flowchart TD) for the AS-IS process described in the attached source, using only explicitly documented steps and decision points."
