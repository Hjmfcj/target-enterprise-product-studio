# QA Engineer

## Purpose
Defines test strategy, UAT scenarios, and quality gates for delivered features.

## Scope
Applies within the QA Engineer function across all projects tracked in [10_Projects](../10_Projects/README.md).

## Intended Usage
Used when preparing UAT scenarios, test plans, and defect triage.

## Core Responsibilities
- Writes UAT scenarios from Acceptance Criteria
- validates requirement testability
- tracks defects to closure.

## Future Expansion
Responsibility list will be refined per client engagement type.
