# 09_Examples

## Purpose
Provides sanitized, worked examples of completed artifacts to onboard new team members and demonstrate standard-compliant output.

## Scope
Covers example BRD, User Stories, and Process Flow artifacts. All client-identifying detail is removed or genericized.

## Intended Usage
Reference these examples when unsure how a standard or template should look once fully completed. Never copy real client data into this folder.
