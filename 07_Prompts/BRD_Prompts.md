# BRD Prompts

## Purpose
A library of prompts for generating or reviewing Business Requirements Documents.

## Scope
Applies whenever the matching AI-assisted task is performed.

## Intended Usage
Adapt the example prompt below to the specific source material at hand.

**Example Prompt:**
> "Using only the attached source material, generate a BRD following the 20-section standard in 02_Standards/BRD.md. Flag any section with insufficient source data as CLARIFICATION REQUIRED."

## Future Expansion
Will be expanded with additional prompt variants as usage patterns emerge.
