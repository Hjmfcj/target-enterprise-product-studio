# 08_Checklists

## Purpose
Provides quality gate checklists that must be run before any deliverable is considered final.

## Scope
Covers BRD, User Story, UAT readiness, and general requirement quality checks.

## Intended Usage
Run the relevant checklist immediately before delivery, and attach the completed checklist result as part of the Gap Analysis section of the deliverable.

## Contents
- [BRD QA Checklist](BRD_QA_Checklist.md)
- [User Story QA Checklist](User_Story_QA_Checklist.md)
- [UAT Readiness Checklist](UAT_Readiness_Checklist.md)
- [Requirement Quality Checklist](Requirement_Quality_Checklist.md)

## Future Expansion
Additional checklists (e.g. Process Flow QA, Screen Inventory QA) will be added as new deliverable types recur.
