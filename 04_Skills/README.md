# 04_Skills

## Purpose
Defines discrete, AI-executable skills — narrow, well-scoped capabilities that an AI assistant applies on demand within this repository's rules.

## Scope
Covers the core BA/PO production skills: discovery synthesis, BRD generation, user story generation, backlog management, process flow generation, and gap analysis.

## Intended Usage
Invoke the relevant skill file as an instruction set when performing the matching task; skills always operate under [00_System/04_Project_Rules.md](../00_System/04_Project_Rules.md).
